Overview

This library provides state-of-the-art machine learning models for natural language processing, computer vision, and multimodal tasks. Built with flexibility and performance in mind, it supports multiple deep learning frameworks while maintaining a consistent API.

Architecture & Core Components

The library is organized around several key components that work together to provide a unified interface for working with various model architectures.

Configuration Classes

Store model parameters and hyperparameters. All configuration classes inherit from PretrainedConfig base class. 

from transformers import AutoConfig

# Load configuration
config = AutoConfig.from_pretrained("bert-base-uncased")
print(config.hidden_size)  # 768

API Reference

AutoModel

Automatically selects the appropriate model class based on the checkpoint.

Parameter Type Description Default
pretrained_model_name_or_path str Model identifier or path required
config PretrainedConfig Model configuration None
cache_dir str Cache directory None
Note: The AutoModel class automatically determines the model architecture from the configuration file in the checkpoint.
